Preparação do ambiente (Voltar)

A principal ferramenta para iniciarmos o curso é o editor visual studio code, basta clicarmos no Link e baixar a ferramenta e instalar no computador.

Uma funcionalidade que devemos ter é o Live Serve, ele é um plugin e para instalar basta ir nas opções de extensões do visual code e instalar.

O Live Serve é utilizado para abrir uma pagina de browser(navegador) que é atualizada de forma automatica enquanto atualizamos nossa pagina

Estrutura basica do HTML (Voltar)

O HTML consite basicamente em 3 partes.

O "HTML"(define a pagina), o "head"(é a cabeça da estrutura, basicamente a mente do site, informações pré-carregadas que não vai ser exibida para o usuario) e o "body"(que é o corpo, local que vai ficar toda a escrita).

TAG's (Voltar)

Existem varias tags e irei colocar alumas para termos

"i" Deixar o texto em italico.
"strong" Deixar o texto negrito.
"br" é utilizada para pular linha.
"input"cria um campo digitabel.
"img" colocar imagem.
"id" é um atributo identificador de varios elementos, é utilizado para dar atributo a algum elemento.
"style" é voltada para o CSS, elemento de estilização.
"class" é uma forma de padronizar uma linha para deixala estilizada.
"type" é utilizado para definir uma espeficicação da tag "imput".
"src" é utilizado para complementar a "img"

Textos (Voltar)

Inicialmente temos os H's, neles temos do H1(Maior) ate o H6(menor), o H1 é utilizado para titulos e o H2 é utilizado para subtitulos.
Temos o paragrafo que usados a tag "p", nela colocamos o texto.
A tag "blockquode" é usada para fazer citações e da um espaçamento no texto.

Listas Ordenadas e Não Ordenadas (Voltar)

Temos 2 listas principais, a primeira é a "ol" e ela começa a lista ordenada e logo em seguida temos que coloca a tag "li" que são as filhas e elas que vão ficar com o texto.
Já a Não Ordenada, ela começa com ul, e logo em seguida temos que colocar a tag li.

Links (Voltar)

Podemos utilizar a tag"a" para colocar um link, ela é uma ancora.
Podemos tambem utilizar a tag target="_blank" para abrir a pagina em outra aba.
